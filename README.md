# HALO

[![Lines of Code](https://tokei.rs/b1/github/andrei-ace/HALO)](https://github.com/andrei-ace/HALO)

HALO is a robotic manipulation system built around **continuous control decoupled from LLM reasoning** — the robot never pauses motion waiting for the planner.

See [docs/README.md](docs/README.md) for the full architecture documentation.
